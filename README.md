Excel Consolidation Project
This project demonstrates a small financial consolidation model built in Excel.
It includes two fictional subsidiaries (SubA and SubB), an intercompany transaction, a manual elimination entry, and a consolidated income statement.
A PivotTable summarizes revenue by subsidiary to show basic reporting and analysis.

Skills demonstrated:
Group accounting logic
Intercompany elimination
Structured financial modeling
Excel formulas & PivotTables
Clean data organization
